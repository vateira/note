Dartでパイプライン演算子的なのを作る

2021-04-26

Dart で F#や Elixir みたいなパイプライン演算子ができるかと思って実験。 Dart には部分的用とか無いので、作ってみたものの実用性は無いか。

ちなみに tc52 ではなく tc39 だと stage 1 に pipeline 演算子が出ているので 、JavaScript はそのうちできるようになるかも。

class Pipeable<T> {
  final T value;
  Pipeable(this.value);

  Pipeable operator >> (Function(T) f)
  {
    return Pipeable(f(value));
  }

}

void main() {
  Pipeable(10)
    >> ((x) => x * x)
    >> ((x) => 'squared value is ${x}')
    >> ((x) => print(x))
    ;
}
