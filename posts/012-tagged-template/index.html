google/zxとJavaScriptのTagged Templates

2021-05-11

社内の雑談でgoogle/zx が話題になった際に、$`find -type f`のような構文について質問があったので。

特殊な構文でプリプロセスしているとかいうのではなく、これは Tagged Templages を使っている。

function hoge(strs, ...args) {}

のような関数を定義して

const va = "[a]";
const vb = "[b]";
hoge`a${va}b${vb}`;

とすると、関数hogeが呼び出され、引数strsに["a", "b"]が、argsに["[a]", "[b]"]が渡される。

zx の場合は

function $(strs, ...args) {
  return new Promise((resolve, reject) => {
    // execなどにより外部コマンドを実行
  });
}

のように関数$を定義しているので、冒頭にあったような

let branch = await $`git branch --show-current`;

のような書き方ができる。

文字列リテラルに対して何らかの関数を適用できると言う点では、Elixir の sigil に似ていると思う。
